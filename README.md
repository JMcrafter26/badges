## Features 🌟

- Very simple 😌
- Many features 🤯
- Lightweight (minified ~4kb) 💪
- Highly customizable 🎨
- Host it yourself! (Or use mine) 🌐
- No setup required, just upload and go 🚀
- More coming soon! 🔜
